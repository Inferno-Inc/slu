## Install

### Mac

```
brew install sikalabs/tap/slut
```

### Autocomplete

See: `slut completion`

#### Bash

```
source <(slut completion bash)
```

## CLI Usage

### Generated CLI Docs on Github

See: <https://github.com/sikalabs/slut-cli-docs/blob/master/slut.md#slut>

## Generate CLI Docs

Generate Markdown CLI docs to `./cobra-docs`

```
slut generate-docs
```
