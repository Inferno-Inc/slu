## Install

### Mac

```
brew install sikalabs/tap/slu
```

### Autocomplete

See: `slu completion`

#### Bash

```
source <(slu completion bash)
```

## CLI Usage

### Generated CLI Docs on Github

See: <https://github.com/sikalabs/slu-cli-docs/blob/master/slu.md#slu>

## Generate CLI Docs

Generate Markdown CLI docs to `./cobra-docs`

```
slu generate-docs
```
